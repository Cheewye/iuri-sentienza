# 🔺 Feature: Visualización Cuántica de Glifos

## Descripción
Un generador simbólico de glifos visuales en 2D y 3D, asociados a escalas, colores y emociones.

## Funcionalidades esperadas
- Asociación entre escala musical y geometría.
- Generación SVG o canvas interactivo (HTML).
- Opción de animación evolutiva.
- Exportar imágenes a disco para visualización en frontend.

## Entradas
- `escala`: nombre o patrón.
- `emoción`: string descriptivo (feliz, tenso, oscuro, expansivo, etc.).

## Salidas
- SVG o archivo canvas.
- Metadata visual asociada.

## Branch sugerido
`feature/symbolic-glifo-generator`

## Directorio sugerido
- Código: `frontend/glifo_generator.js`
- Visuales: `frontend/assets/glifos/`
